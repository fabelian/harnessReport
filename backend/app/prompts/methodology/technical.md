# Technical Analysis Methodology (MVP 발췌)

기술적 분석 시 사용하는 지표 정의와 해석 가이드.

## 1. 추세 진단
- **이동평균선 정배열**: MA5 > MA20 > MA60 > MA120 > MA200 → 강한 상승 추세
- **역배열**: 위 순서 역전 → 하락 추세
- **혼조**: 정·역 섞임 → 박스권 / 추세 전환 의심
- 골든크로스: MA단기가 MA장기를 위로 돌파 (강세)
- 데드크로스: 반대 (약세)

## 2. 모멘텀
- **RSI(14)**:
  - > 70: 과매수 (조정 가능성)
  - < 30: 과매도 (반등 가능성)
  - 50선 돌파: 추세 강화 신호
- **MACD(12,26,9)**:
  - MACD > 0 & MACD > Signal: 상승 모멘텀
  - 다이버전스(가격 신고가, MACD 하락): 추세 약화 경고

## 3. 변동성·밴드
- **Bollinger Bands (20, 2σ)**:
  - 상단 터치: 단기 과열
  - 하단 터치: 단기 과매도
  - 스퀴즈(폭 좁아짐) → 변동성 폭발 임박
- **ATR(14)**: 평균 진폭, 손절/익절 기준 설정에 사용

## 4. 거래량
- 가격 상승 + 거래량 증가: 추세 신뢰도 ↑
- 가격 상승 + 거래량 감소: 약한 상승 (조정 우려)
- 가격 하락 + 거래량 폭증: 패닉 / 바닥 시그널

## 5. 시나리오 작성 3요소
1. **트리거**: 가격·지표 조건 (예: "MA50 하향 이탈 + RSI 50 미만")
2. **타깃**: 1차/2차 가격 (예: "1차 $850 (전 저점), 2차 $780 (MA200)")
3. **무효화 조건**: 시나리오가 깨지는 가격 (예: "$920 이상 회복 시 무효")

## 6. 한계 명시 (보고서 말미)
- 후행성: 지표는 과거 데이터를 가공한 결과
- 자기실현적 예언: 다수가 보는 매물대는 실제로 작동하기도, 무시되기도 함
- 거시·이벤트 충격(어닝, 매크로) 발생 시 기술적 시그널은 무력화될 수 있음
