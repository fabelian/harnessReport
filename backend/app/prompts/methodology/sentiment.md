# Sentiment Methodology (MVP 발췌)

시장 심리·뉴스·수급 분석 시 사용하는 프레임워크.

## 1. 신호 강도 (SentimentSignal.strength)
- **strong**: 다수 출처가 같은 방향 + 가격 반응 확인
- **moderate**: 단일 주요 출처 또는 혼재된 신호
- **weak**: 정황만 있고 가격 반응 부재

## 2. 뉴스 신호 (news_signals)
- 어닝 서프라이즈, 가이드 변경
- 신규 계약(공급)·수주, 인증 통과
- 임원 발언, 컨퍼런스 콜 톤
- 규제·정책 (관세, 보조금)

각 항목에 `evidence` 짧게(1줄) + `direction` + `strength`.

## 3. 수급 신호 (flow_signals)
- 외인/기관/개인 누적 순매수 (한국)
- 옵션 풋콜 비율
- 공매도 잔고 (한국 시장 정책 반영)
- 거래량 급증·감소

## 4. 컨센서스
- 12M 목표주가 평균·범위·기관 수 (출처 명시)
- 2026E EPS 컨센 추세 (상향/하향)
- 투자의견 분포 (매수/보유/매도)

컨텍스트에 컨센 데이터 없으면 `consensus_note`에 "본 분석에서는 미제공 — 미확보"로.

## 5. 종합 톤 (overall_tone)
- **bullish**: 신호 다수 양(+)
- **neutral**: 혼재
- **bearish**: 신호 다수 음(-)

## 6. 인용 규칙
- 검증 불가 SNS·루머 인용 금지
- 컨텍스트의 `news.items`만 사용, URL 보존
- 단정적 추천 금지
