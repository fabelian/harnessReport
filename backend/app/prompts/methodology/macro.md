# Macro Methodology (MVP 발췌)

거시 환경 분석 시 점검하는 변수.

## 1. 핵심 변수
| 변수 | 출처 | 주가에 미치는 채널 |
|------|------|------------------|
| Fed Funds Rate | FRED FEDFUNDS | 멀티플(고PER 종목 민감) |
| 10Y UST | FRED DGS10 | 할인율, 성장주 멀티플 |
| Trade-Weighted USD | FRED DTWEXBGS | 수출주 매출 환산 |
| VIX | FRED VIXCLS | 위험 선호 |
| 한·미 금리차 | BOK ECOS + FRED | 외인 자금 흐름 |

## 2. FX 채널 (한국 종목)
- USD 강세 → 한국 수출주(반도체 포함) 매출 환산 ↑, 단기 호재
- USD 약세 → 외인 한국 비중 확대 가능성

## 3. CapEx 사이클
- 빅테크 (MSFT·GOOGL·META·AMZN·ORCL) capex 가이드 → AI 인프라 수요 → HBM·서버 메모리
- 가이드 상향 = tailwind, 하향 = headwind

## 4. 상관관계 (정성)
- 메모리주 ↔ SOX/엔비디아: 강한 양의 상관
- 메모리주 ↔ 한국 KOSPI: 시총 비중 큰 종목은 지수에 영향

## 5. 시나리오 바이어스
- **Tailwind**: 디스인플레 + 완만한 성장 + AI capex 지속
- **Neutral**: 금리 횡보 + 수요 보합
- **Headwind**: 금리 재인상 + 침체 시그널 + capex 가이드 하향

## 6. 단정적 예측 회피
환율·금리는 단정 표현 금지 — "X 방향 전망 강화" 정도로.
