# Valuation Methodology (MVP 발췌)

펀더멘털·밸류에이션 분석 시 표준 산식과 해석 가이드.

## 1. 핵심 멀티플

| 지표 | 산식 | 해석 포인트 |
|------|------|------------|
| PER (TTM) | 시가총액 / 최근 4분기 순이익 | 사이클 정점에서 낮고 바닥에서 높을 수 있음 — 정상화 EPS 별도 산출 |
| PER (Fwd) | 시가총액 / 12M Fwd EPS | 컨센서스 EPS 사용, 출처 명시 |
| PBR | 시가총액 / 순자산 | 반도체·메모리는 자본집약형으로 PBR 의미가 큼 |
| EV/EBITDA | 기업가치 / EBITDA | 차입 차이 통제 — 글로벌 동종 비교에 유용 |
| EV/Sales | 기업가치 / 매출 | 매출 변동 큰 사이클 산업에 보조 |
| Dividend Yield | DPS / 주가 | 메모리는 사이클로 인해 안정성 낮음 |

## 2. 동종 비교 (반도체)
- 미국: NVIDIA(NVDA), AMD, Broadcom(AVGO), Micron(MU)
- 한국: 삼성전자(005930), SK하이닉스(000660)
- 일본: Kioxia(285A.T)

비교 시 사이클 정렬(같은 사이클 국면) 확인.

## 3. 시나리오 작성 3요소
모든 시나리오는 반드시 다음을 포함:
1. **트리거**: 가격·실적·사이클 조건 (예: "DRAM 현물가 3분기 연속 상승")
2. **가정값**: 멀티플·성장률·할인율 (예: "Fwd PER 12x, 매출 +20% YoY")
3. **결과 범위**: 적정가 low~high (단정적 값 금지)

가정값이 빠진 시나리오는 검증 불가능 → 작성하지 않는다.

## 4. 메모리 사이클 주의사항
- PER 함정: 정점에서 EPS 폭증 → PER 낮음 (저평가 아님)
- 정상화 EPS: 사이클 평균 OP 마진 적용
- HBM은 일반 DRAM 대비 ASP·마진 프리미엄 ("AI 메모리 프리미엄")

## 5. SOTP (Sum-of-the-Parts) 간이 절차
1. 부문별 매출 분해 (공시된 범위 내)
2. 각 부문에 동종 멀티플 적용
3. 합산 후 부채/현금 조정 → 적정 시가총액
4. 부문별 OP 미공시 시 "비공개 — SOTP 부분 수행 불가" 명시
