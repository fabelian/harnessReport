# Citations & Disclaimers — 인용·면책 표준 (Web App 버전)

본 분석에서 모든 에이전트가 따르는 출처 표기·면책 규정.

## 1. 인용 우선순위
- **1차 자료**: 회사 공시 (SEC EDGAR / DART), 회사 IR 자료, 거래소(KRX·NASDAQ) 공식 데이터, 중앙은행·정부 통계 (FRED, BOK ECOS)
- **2차 자료**: 주요 언론 (Reuters, Bloomberg, WSJ, 한경, 매경), 증권사 리포트 — 1차 자료로 역추적 후 인용
- **3차 자료 (인용 자제)**: 익명 SNS, 검증 불가 블로그, 위키피디아

본 분석은 사전 수집된 데이터 컨텍스트(`AnalysisData`)에 한정한다. 컨텍스트에 없는 외부 출처를 임의로 인용하지 않는다. 만약 인용할 자료가 있다면 컨텍스트의 `news.items` 또는 `data` 객체 안의 명시된 출처만 사용한다.

## 2. 사실 / 추정 / 의견 3단 라벨링

모든 주요 주장은 다음 중 하나로 라벨된다 (`label` 필드):
- **fact**: 컨텍스트의 정량 데이터 또는 공시·통계에서 확인 가능
- **estimate**: 컨센서스·가이던스·시장조사기관 전망 (출처 표기 필수)
- **opinion**: 분석가의 판단 (예: "본 분석에서는…")

라벨이 모호한 문장은 보고서에 포함하지 않는다.

## 3. 수치 표기 규칙
- 단위와 시점을 항상 동반: "Revenue $26.0B (FY24 Q4, source: company filing)"
- 통화 환산이 필요할 때 환산 시점·환율 명시
- 비율은 소수점 둘째 자리까지 통일

## 4. 단정적 표현 금지
- "주가가 X로 갈 것이다" → "현재 멀티플 기준 적정가 X~Y 범위"
- "반드시 매수" → 시나리오·트리거·범위로 제시

## 5. 데이터 미확보 시 처리
- 임의 추정값 생성 금지
- `data_caveats[]`에 "확인 필요" 또는 "X 미확보 (source: ...)"로 명시
- 정성 평가로 보완

## 6. 면책 조항 (최종 보고서 말미)

```
## 면책 조항
본 보고서는 공개된 정보를 바탕으로 작성된 정보 제공 목적의 분석 자료이며,
특정 주식의 매수·매도·보유를 권유하는 투자권유 자료가 아닙니다.
미래의 실제 결과와 다를 수 있으며, 투자 결정은 본인의 책임 하에 이루어져야 합니다.
```
